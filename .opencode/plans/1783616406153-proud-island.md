## What the pipeline does

| Job | Runs on | What it does |
|---|---|---|
| `test` | macOS | `bun install` → `bun run test` → `bun run test:smoke` → `bun run build` |
| `release` (matrix) | macOS/Windows/Linux | `electron-builder` → packages into `.dmg`, `.zip`, `.exe`, `.AppImage`, `.deb`, `.rpm` |
| `publish` | Ubuntu | Creates GitHub Release, uploads all platform artifacts |
