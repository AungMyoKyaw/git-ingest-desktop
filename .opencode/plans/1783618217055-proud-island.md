# GitHub Actions Version Upgrade Plan

**Goal:** Update all GitHub Actions in CI/CD workflows to latest versions that run on Node.js 24+ (fix deprecation warnings).

**Architecture:** Three workflow files — `ci.yml`, `release.yml`, `pages.yml` — all using old action versions (`@v4`) that run on deprecated Node.js 20.

## Current → Target

| Action                      | Current | Target | Node runtime |
| --------------------------- | ------- | ------ | ------------ |
| `actions/checkout`          | `@v4`   | `@v7`  | node24       |
| `actions/setup-node`        | `@v4`   | `@v6`  | node24       |
| `actions/upload-artifact`   | `@v4`   | `@v7`  | node24       |
| `actions/download-artifact` | `@v4`   | `@v8`  | node24       |

## Files to Modify

- `.github/workflows/ci.yml` — 2 occurrences: checkout, setup-node
- `.github/workflows/release.yml` — 5 occurrences: checkout (2x), setup-node (2x), upload-artifact, download-artifact
- `.github/workflows/pages.yml` — 1 occurrence: checkout

## Steps

1. **Update all `actions/checkout@v4` → `actions/checkout@v7`** in all 3 files
2. **Update all `actions/setup-node@v4` → `actions/setup-node@v6`** in ci.yml and release.yml
3. **Update `actions/upload-artifact@v4` → `actions/upload-artifact@v7`** in release.yml
4. **Update `actions/download-artifact@v4` → `actions/download-artifact@v8`** in release.yml
5. **Verify** — push and confirm no more Node.js 20 deprecation warnings in Actions logs

## Verification

Push to master (or trigger workflow_dispatch) and check Actions run output — no "Node.js 20 is deprecated" warnings should appear.
